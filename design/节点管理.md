# 节点管理

## 节点列表
```graphql
type Label {
  key: String! # 标签键
  value: String! # 标签值
}
type Node {
  ip: String!
  port: String
  username: String!
  password: String!
  labels: [Label!]! # 标签列表
}
type NodeList {
  nodes: [Node!]!
}
type AddNodeInput {
  ip: String!
  port: String
  username: String!
  password: String!
  labels: [Label!]! # 标签列表
}
type UpdateNodeInput {
  ip: String!
  port: String
  username: String!
  password: String!
  labels: [Label!]! # 标签列表
}
type DeleteNodeInput {
  ip: String!
}
mutation AddNode($input: AddNodeInput!) {
  addNode(input: $input) {
    ...NodeList
  }
}
mutation UpdateNode($input: UpdateNodeInput!) {
  updateNode(input: $input) {
    ...NodeList
  }
}
mutation DeleteNode($input: DeleteNodeInput!) {
  deleteNode(input: $input) {
    ...NodeList
  }
}
```
node信息存储在/var/lib/ar/nodes/目录下,文件名为node_ip.json,文件内容为节点IP、端口、用户名、密码。
